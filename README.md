## How It Works

1. Rider opens the app and sets pickup and drop-off locations with RTL interface
2. Nearest captain receives a trip request with fare preview
3. Captain accepts and navigates to the pickup location
4. Trip starts; fare is calculated in real time based on distance and time
5. Payment is processed automatically from wallet or card
6. Trip completes; both parties rate each other
